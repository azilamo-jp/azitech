I like using Hugo for a blog because it keeps the writing workflow simple.
With a minimal theme, there is less overhead and more room to focus on the actual content.

## What works well

1. Builds are fast.
2. Writing in Markdown keeps publishing straightforward.
3. The structure and styling can evolve gradually over time.

## This setup

1. The home page introduces the site and shows recent posts.
2. `/blog/` contains the full post archive.
3. Each post uses a simple layout focused on readability.

For now, I want to keep the site lightweight and only add features when they become clearly useful.
